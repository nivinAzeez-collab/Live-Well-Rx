# Live Well Pharmacy Mobile App
## Product Requirements Document (PRD)

---

# 1. Overview

**Product Name:** Live Well Pharmacy Patient App  
**Company:** Live Well Pharmacy  
**Location:** 1722 Utica Ave, Brooklyn, NY

## Purpose

The Live Well Pharmacy app will allow patients to manage their prescriptions digitally. Patients will be able to refill prescriptions, request additional refills, choose delivery or pickup, and communicate with the pharmacy.

The goal is to improve patient convenience, increase prescription adherence, reduce phone calls, and grow pharmacy revenue through digital engagement.

---

# 2. Goals

## Primary Goals

- Allow patients to **refill prescriptions quickly**
- Allow patients to **request additional refills**
- Allow patients to **choose delivery or pickup**
- Reduce pharmacy **phone call volume**
- Increase **patient retention**
- Expand **digital pharmacy services**

---

# 3. Target Users

## Primary Users

- Live Well Pharmacy patients
- Elderly patients
- Patients with chronic conditions
- Caregivers managing medications

## Secondary Users

- Pharmacy staff
- Doctors responding to refill requests

---

# 4. Core Features

---

# 4.1 User Account

Patients must be able to create and manage an account.

## Requirements

- Sign up using:
  - Phone number
  - Email
  - Date of birth
- SMS verification
- Secure login
- Link account to pharmacy patient profile

## User Flow

1. Download app
2. Create account
3. Verify phone number
4. Connect account to Live Well Pharmacy patient record

---

# 4.2 Prescription Refill

Patients should be able to refill prescriptions easily.

## Requirements

Patients can:

- View current prescriptions
- Select medication
- Tap **Refill**

## Refill Options

- Pickup
- Delivery

## Data Displayed

- Prescription number
- Medication name
- Dosage
- Quantity
- Last refill date
- Remaining refills

---

# 4.3 Request Additional Refills

If a prescription has no refills remaining, patients can request authorization.

## Requirements

- Tap **Request Refill Authorization**
- Pharmacy receives request
- Pharmacy contacts doctor
- Patient receives status updates

## Notifications

Patients receive notifications when:

- Request is sent
- Doctor approves refill
- Prescription is ready

---

# 4.4 Delivery or Pickup Selection

Patients choose how to receive medications.

## Delivery Options

- Home delivery
- Same-day delivery (local)
- Mail delivery

## Pickup Options

- In-store pickup
- Scheduled pickup time

---

# 4.5 Prescription Status Tracking

Patients can track prescription progress.

## Status Types

- Received
- Processing
- Waiting for doctor approval
- Ready for pickup
- Out for delivery
- Completed

---

# 4.6 Notifications

Push notifications keep patients informed.

## Notification Types

- Prescription ready
- Doctor approved refill
- Delivery on the way
- Refill reminder
- Medication reminder

---

# 4.7 Medication List

Patients can view their medications.

## Features

- Active medications
- Past medications
- Dosage instructions
- Refills remaining

---

# 4.8 Secure Messaging

Patients can communicate with the pharmacy.

## Features

- Secure HIPAA-compliant messaging
- Ask pharmacist questions
- Medication support

---

# 4.9 Delivery Tracking

Patients receiving delivery can track status.

## Features
